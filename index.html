Malaysian Food Menu Translator Embedded with AR

This application have augmented Reality display the 3D food model in the real environment, text recognition for the user to capture the food name in the real world, and view the food detail to get more information about the food. Moreover, Malaysian Food Menu Translator must make users to better understand food characteristics and reduce language translation problems. The project will develop a text recognition application with augmented reality technology to translate the food name from Malay into English and 3D models.

  • English language for application interface, Malay is the main translation language to English.
  • The main target food is Malaysian.
  • Android Studio, Java programming, Google Vision and AR Core will be used in the development.
  • The application is targeted at non-local users who do not know the Bahasa language.
